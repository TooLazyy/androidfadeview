### Usage
---

Just put views you need to fade inside
```
    <ru.wearemad.fadeview.FadeView
        android:layout_width="match_parent"
        android:layout_height="match_parent">

<!--your view here-->
</ru.wearemad.fadeview.FadeView>
```
